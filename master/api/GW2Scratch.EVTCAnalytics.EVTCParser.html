ParseLog(String)

Reads raw data from an EVTC log file.

Declaration
public ParsedLog ParseLog(string evtcFilename)
Parameters
Type Name Description
String evtcFilename

The filename of the log.
Returns
Type Description
ParsedLog

The raw data from the log.
Remarks

Note that the extension is used to identify whether this is a zipped EVTC log. If the filename ends with .zip or .zevtc, it is opened as a zip file.

If you do not need access to the ParsedLog, using the ProcessLog(String, EVTCParser) method is more efficient.

Exceptions
Type Condition
NotSupportedException

Thrown for unsupported log revisions.
LogParsingException

Thrown when parsing fails due to a malformed log or if an empty ZIP archive is supplied.
