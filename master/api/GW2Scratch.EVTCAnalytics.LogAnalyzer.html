Class LogAnalyzer

GetMainEnemyHealthFraction()

Calculates the final health percentage of the main enemy or enemies in an encounter.

Declaration
public float? GetMainEnemyHealthFraction()
Returns
Type Description
Nullable<Single>

A non-negative health fraction value where 1 corresponds to 100% or null if not applicable for this encounter.
Remarks

This value may have different semantics depending on the encounter. For example, for encounters with multiple boss enemies, the higher of their health values may be used.

The returned value may be higher than 1 in case there are multiple enemies in sequence in the encounter, each extra 1 corresponds to a health bar in that case.

This method caches its result, calculating the result only once.
