Class Agent

Represents an agent in the game - an entity with health, capable of using skills and movement in some cases.

Remarks

A single Agent may correspond to multiple raw ParsedAgents. The AgentOrigin may be used to see where it comes from. This mechanism is required to handle merging NPCs that leave the reporting area.
