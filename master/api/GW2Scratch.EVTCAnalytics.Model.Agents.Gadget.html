Class Gadget

Represents a gadget in the game. Gadgets are a special type of NPCs that are immobile and can be damaged through AttackTargets.

Remarks

Not all combat events are available for gadgets. They are also exempt from maximum distance limits for reporting data, so logs often contain gadgets located in a far away part of a map.
